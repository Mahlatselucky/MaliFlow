# Mali Flow💰

> *Know where your mali goes*

A personal budget tracking Android app built with Kotlin and RoomDB for OPSC6311.

## Features

-  User login and registration
-  Create and manage expense categories
- Add expense entries with date, time, description and amount
-  Attach photos to expense entries
-  Set monthly minimum and maximum budget goals
-  View expense entries for a user-selected period
-  View category totals for a user-selected period

## Tech Stack

- Kotlin
- Android Studio
- RoomDB (local SQLite database)
- Material Design Components
- MVVM Architecture
- Glide (image loading)

## How to Run

1. Clone the repo
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or physical device (API 26+)
